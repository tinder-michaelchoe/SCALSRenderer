# Document Schema Changelog

All notable changes to the SCALS Document schema (JSON API) will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-02

### Added
- Initial Document schema release
- Core components: Container (VStack, HStack, ZStack), Text, Button, Image, TextField
- Style system with named styles and inline properties
- Action binding system for component interactions
- Padding shorthand syntax (horizontal, vertical, all, individual edges)
- Color support (hex, named colors, system colors)
